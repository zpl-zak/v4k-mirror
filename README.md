This repository is a mirror of our public build of the game engine.

## Release Structure

- **Latest Version**: The `latest` folder contains the most up-to-date mirror
  of our internal VCS. This may include unreleased changes and might pose instability issues. USE AT YOUR OWN RISK. Latest build can be found here: [v4k-mirror](https://github.com/zpl-zak/v4k-mirror)

- **Stable Releases**: Found in folders named after their version (e.g., `2024.9`).
  These are considered stable and contain specific release versions. Releases are available here: [v4k-archive](https://github.com/zpl-zak/v4k-archive).

## Changelog

For information on releases, please refer to the `CHANGELOG.md` file in respective release folder.
