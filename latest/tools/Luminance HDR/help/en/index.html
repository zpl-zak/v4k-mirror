Welcome to Luminance HDR!

Luminance HDR is an open source workflow tool for HDR imaging.

Its highlights are:

  • streamlined user interface in more than a dozen languages
  • support for all major HDR file formats
  • batch hdr creation
  • batch tonemapping
  • resize/crop/transform HDRs
  • color managed workflow
  • works on Linux, Windows and Mac OS X
  • free to distribute among family, friends and enemies

If you are new to HDR imaging, please read the introductional chapter. If you already know what's good in HDR for you, you might like reading detailed features overview. Finally, if you know your way around Luminance HDR, you can learn what's new in this version.
